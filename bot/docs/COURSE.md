# Комментарии к учебному тексту

Справочник к `course_from_glossed3.json`: что означают поля, откуда взяты формулировки и на что обратить внимание при вычитке.

## Мета-блок (`meta`)

| Поле | Смысл |
|------|--------|
| `course` | Название курса для авторов |
| `source` | Устный корпус + отдельный вводный блок по слову «аз» |
| `structure` | Блок 1 — падежи на «аз»; далее темы: лексика → падежи → (с темы 4) предложения |
| `case_inventory` | Полный список падежных ярлыков в JSON |
| `case_note` | Ine отдельно от генитива; комитатив в темах не используется |
| `exercise_types` | Словарь типов заданий в исходнике → см. маппинг в `regenerate_seeds.py` |
| `distractor_note` | Для перевода дистракторы не фиксируются в JSON |
| `audio_note` | `audio: null` — слот под запись; имя файла = `{word_id}.ogg` |
| `quality_note` | `needs_validation: true` — нужна проверка носителем |
| `verbs_note` | Глаголы без падежных drills, только формы времени |

## Блок (тема)

- `id` — номер темы (1–6).
- `title` / `title_os` — заголовок RU / OS; `title_needs_validation` — флаг вычитки.
- `words` / `verbs` — лексемы с парадигмой или формами времени.
- `exercises` — последовательность заданий уровня до нарезки на 4–5 levels.

## Слово (`words[]`)

| Поле | Комментарий |
|------|-------------|
| `id` | Стабильный ключ: `b1_01`, `t2_03` — совпадает с именем аудио |
| `lemma` | Словарная форма на осетинском |
| `trans_ru` | Перевод; может содержать несколько значений через `;` |
| `paradigm[]` | Падеж, форма, окончание, `function` — текст для карточек правил |
| `distractor_pool` | Ложные варианты для перевода (lemma + trans_ru) |
| `audio` | Имя файла в `assets/audio/`, например `t2_01.ogg` |
| `needs_validation` | Автоизвлечение из корпуса — требует проверки |

### Пример: блок 1, слово «аз»

В корпусе «аз» размечено как существительное «год» — учебный приём: на одном частотном слове показать все падежи, прежде чем переходить к тематической лексике.

## Типы упражнений в JSON → бот

| JSON | В боте | Текст для пользователя |
|------|--------|------------------------|
| `rule_card` | `theory` | «Падеж … (окончание): функция. Пример: …» |
| `learn_word` | `vocab` | «Новое слово … Нажмите «Далее»» |
| `translate_word` | `translate_os_ru` | «Переведите на русский: …» |
| `choose_case` | `choice` | «Какой падеж у формы …» |
| `choose_form` | `choice` | «Нужен падеж … Выберите форму» |
| `match_case` | несколько `choice` | «Парадигма … укажите падеж формы» |
| `assemble_sentence` | `translate_ru_os` | «Соберите предложение … слова: …» |

## Темы курса

1. **Введение в падежи** — «аз», сахар, район, мыггаг; полная парадигма на вводном слове.
2. **Семья и люди** — лӕг, сывӕллон, лӕппу, чызг и др.
3. **Дом и быт** — предметы дома + первые глаголы.
4. **Время и день** — время суток, появляются `assemble_sentence`.
5. **Село и природа** — хъӕу и природная лексика.
6. **Язык и обычай** — культурный контекст, ӕвзаг и др.

## Озвучка (на момент написания)

Файлы с аудио: `b1_01`, `b1_03`, `b1_04`, `b1_05`, `t2_01`–`t2_04`, `t5_01`, `t6_02`.

Правило именования: `{word_id}.ogg` в `assets/audio/`. После добавления файла:

1. Указать `"audio": "{word_id}.ogg"` в `words[]` и в `learn_word` (если есть).
2. Запустить `python scripts/regenerate_seeds.py`.
3. Пересобрать бота.
