# Осетинский язык: duolingo-бот

Telegram-бот для изучения осетинского языка в формате Duolingo.

Бот учит словам и падежам осетинского: показывает слово с переводом,
проверяет перевод, тренирует падежные формы, а на старших уровнях даёт
собирать предложения из слов.

## Как это работает

```
корпус ──▶ авторазметка ──▶ таблица для носителя ──▶ оценка качества
                                                            │
              разметка оказалась ненадёжной ◀───────────────┘
                          │
                          ▼
        глоссированный корпус ──▶ курс для бота ──▶ Telegram-бот
```

Сначала мы хотели опереться на автоматическую разметку корпуса. Но проверка
носителем показала, что она ненадёжна (ошибки как раз в важных местах —
падежи, времена, отглагольные существительные). Поэтому курс собираем из
**глоссированного корпуса**, где разметка ручная и качественная.

## Как запустить

### Пайплайн разметки

Автоматическая морфоразметка корпуса (файл `corpus.conllu` — с Яндекс Диска):

```bash
python annotation_pipeline/annotate.py --input corpus.conllu
```

Построить таблицу для проверки носителем:

```bash
python annotation_pipeline/build_review_sample.py
```

Оценить качество разметки (после проверки носителем):

```bash
python annotation_pipeline/score_review.py annotation_pipeline/review_sample.xlsx
```

Собрать курс для бота (пути к `glossed.json` и `block1_words_cases.json` подставляются автоматически):

```bash
python annotation_pipeline/build_course.py \
    --output bot/course_from_glossed.json
```

## Как устроен курс

Вводный блок, дальше — темы. Внутри каждой темы:

1. **Учим слова** — карточка со словом и переводом (+ озвучка),
   потом выбор перевода из четырёх вариантов.
2. **Учим падежи** — по уже выученным словам: показать форму и назвать падеж;
   показать словарную форму и нужный падеж — выбрать правильную форму.
3. **Собираем предложения** (с 4-й темы) — дан перевод, нужно собрать
   осетинское предложение из слов. Для разнообразия с 3-й темы добавляются глаголы.

Типы заданий: `theory`, `vocab`, `choice`, `translate_os_ru`, `translate_ru_os`.
